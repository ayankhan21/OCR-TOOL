TESSERACT IS AN OPEN SOURCE TEXT RECOGNITION (OCR) ENGINE, AVAILABLE UNDER THE APACHE 2.0 LICENSE. IT CAN BE USED DIRECTLY, OR USING AN API TO EXTRACT PRINTED TEXT FROM IMAGES.

IMAGE PRE PROCESSING IS A CRUCIAL PART IN OPTICAL CHARACTER RECOGNITION

STEPS INVOLVED IN IMAGE PREPROCESSING ARE

1) GREY-SCALING     2) SKEW CORRECTION     3) NOISE REMOVAL     4) THINNING AND DOUBLE STRUCTURING
